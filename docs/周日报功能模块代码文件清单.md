|---|---|

前端路由注册了以下路径：
- `/daily-report` → 日报详情
- `/daily-reports` → 日报列表
- `/weekly-report` → 周报详情
- `/weekly-reports` → 周报列表
- `/center-daily-report` → 中心日报
- `/weekly-report-review/:id` → 周报评审详情
