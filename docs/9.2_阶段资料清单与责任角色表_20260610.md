## 1. 编制说明

本文件为 20260610 版阶段资料清单正式文档，用于替代旧版 `docs/9.2_阶段资料清单与责任角色表.md` 作为后续程序运行资料模板依据。旧版 48 项资料模板直接废弃，不再作为新项目初始化、齐套摘要或阶段推进门禁的依据。

当前项目仍处于开发阶段，旧模拟项目数据不做兼容，不做历史项目补初始化，不做旧资料项迁移，不做资料模板 `v1` / `v20260610` 共存。

## 2. 字段说明

| 字段 | 说明 |
|---|---|
| 资料编号 | 20260610 版资料项编号 |
| 资料项 | 来自 20260610 流程图右侧“产出”列 |
| 是否必需 | “是”表示影响齐套摘要和阶段推进门禁；“建议”表示展示但不影响齐套摘要和阶段推进门禁 |
| 提交方式 | 当前系统能力下建议的提交方式，可为在线表单、文件上传或在线表单+文件上传 |
| 默认责任中心/角色 | 根据流程图泳道和责任说明整理 |
| 确认/组织角色 | 负责组织、确认或最终收口的角色 |
| 文件平台目标目录 | 第一版保存为 `targetFolderPath`；目录 ID `targetFolderId` 继续为空 |
| 备注 | 来自流程图或与旧版差异说明 |

文件平台目标目录统一采用：

```text
<阶段序号>-<阶段名称>/<资料编号> <资料项>
```

示例：

- `1-立项/1.1 项目需求表`
- `3-合同签订/3.3 项目启动书`
- `5-生产制作/5.1 比价表`
- `6-预验收/6.7 预验收单`
- `7-终验收/7.3 终验收单`

## 4. 系统落地口径

- 后端模板版本使用 `v20260610`。
- `EXPECTED_STAGE_DOCUMENT_ITEM_COUNT` 按本正式文档实际资料项数量设置为 54。
- 新建项目直接初始化本正式文档中的 54 项资料项。
- 旧版 48 项资料模板废弃，不做旧模拟项目兼容、不做历史项目补初始化、不做模板共存。

- 文件平台目标目录保存为 `targetFolderPath`；`targetFolderId` 第一版继续为空。
- 本文档不引入日报周报、文件管理平台真实联动、消息提醒或复杂审批流。
