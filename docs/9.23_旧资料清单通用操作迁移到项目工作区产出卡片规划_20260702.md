# 9.23 旧资料清单通用操作迁移到项目工作区产出卡片规划

日期：2026-07-02

## 1. 背景

当前项目详情页已经具备项目工作区 shell：左侧 8 阶段导航、阶段内蓝色模块、上方产出卡片，以及下方旧资料清单兼容区。上一阶段实现只把非立项产出卡片的处理入口定位到下方旧资料清单，通用上传、提交、审核、退回等执行能力仍由旧资料清单承载。

本规划用于定义下一步迁移边界：把现有 20260625 64 项旧资料清单中的通用功能和内容逐步迁移到上方项目工作区产出卡片。当前 change 只把下方旧资料清单降级为兼容区，不隐藏、不删除旧清单组件；隐藏或删除必须由后续独立 change 评估。旧清单清理完成后，后续独立 change 才考虑 v20260629 71 项目标模板默认启用。

## 2. 当前 change 目标

`migrate-stage-document-common-actions-to-workspace-cards-v1` 的目标是规划并约束第一步迁移：让上方产出卡片成为现有项目资料通用操作的主入口。

本 change 第一版只覆盖当前已经落库的项目资料，不新增 71 项候选资料记录，不切换新项目默认模板，不迁移旧项目，不新增数据库结构，不引入复杂流程引擎。

## 3. 三阶段路线

### Phase 1：上方产出卡片承载旧资料清单通用操作

- 上方产出卡片读取现有资料的同一套后端状态、权限、责任人、附件和阻塞原因。
- 产出卡片提供责任人展示、责任人分配/清空、附件列表、附件上传、附件下载、附件删除、提交资料、审核通过、审核退回、返工重提/返工完成、标记不适用、恢复适用等通用能力。
- 立项 `1.1 / 1.2 / 1.3` 继续使用现有在线表单和 `1.2` 专用评价审批，不回退为普通文件上传。
- 无现有资料绑定的 v20260629 候选产出仍只能作为 shell 或候选规划展示，不创建实际资料记录，不展示虚假的可执行按钮。

### Phase 2：下方旧资料清单降级并逐步清理

- 迁移前，下方旧资料清单继续可操作，保证当前业务不中断。
- 迁移中，上方产出卡片作为唯一主操作入口，下方旧资料清单显示辅助/兼容提示，避免形成两个主入口。
- 对已迁移资料，下方旧资料清单不再显示责任人分配/清空、附件上传、附件删除、提交资料、审核通过、审核退回、返工重提/返工完成、标记不适用、恢复适用等主操作按钮。
- 下方旧资料清单可以保留资料状态、责任人只读展示、附件摘要/附件只读列表、阻塞原因、兼容提示和“到上方产出卡片处理”的定位入口。
- 本 change 只允许把旧资料清单降级为兼容区，不隐藏、不物理删除旧清单组件，也不完全移除旧清单入口。
- 隐藏或删除旧资料清单必须通过后续独立 change 处理，并且以前置覆盖验收通过为条件。

### Phase 3：切换 v20260629 71 项模板

- 只有在上方产出卡片完成通用能力迁移、下方旧资料清单完成降级/清理后，才进入 v20260629 71 项目标模板默认启用评估。
- 71 项模板切换必须通过后续独立 change 明确模板版本、旧项目迁移策略、回滚策略和验收范围。
- 本 change 不把新增 71 项候选落库，不补初始化旧项目，不改变当前 20260625 64 项运行模板。

## 4. 第一版迁移范围

第一版只迁移当前已存在项目资料的通用能力，并且只复用现有接口、权限、状态字段、业务日志和阶段推进判断。

第一版明确不做：

- 不新增资料表结构。
- 不写 migration。
- 不创建第二套上传、提交、审核、退回规则。
- 不创建第二套资料完成状态。
- 不新增复杂流程引擎。
- 不把新增 71 项候选落库。
- 不切换新项目默认模板。
- 不迁移旧项目。
- 不改变阶段推进、工作台任务、业务日志、项目编号规则。
- 不隐藏、不物理删除旧资料清单组件。

## 5. 第一批迁移功能

- 责任人展示。
- 责任人分配/清空。
- 附件列表。
- 附件上传。
- 附件下载。
- 附件删除。
- 提交资料。
- 审核通过。
- 审核退回。
- 返工重提/返工完成。
- 标记不适用。
- 恢复适用。
- 阻塞原因展示。
- 权限原因展示。
- 状态说明展示。

## 6. 立项阶段保护

`1.1 项目需求表`、`1.2 项目立项审批表`、`1.3 项目立项通知` 继续保持现有在线表单和 `1.2` 专用评价/审批能力。

本迁移不得把立项在线表单回退成普通文件上传，不得新增普通提交入口绕过在线表单，不得降低现有评价审批、返工和前置门禁要求。

## 7. 验收重点

- API check 通过。
- Web build 通过。
- OpenSpec validate 通过。
- 浏览器/人工验收上方产出卡片可完成通用资料操作。
- 下方旧资料清单不再作为主入口。
- 已迁移资料在下方旧资料清单不再显示责任人分配、上传、提交、审核、退回、返工、不适用等主操作按钮。
- 本 change 未隐藏、未物理删除旧资料清单组件。
- 立项在线表单不回退。
- 71 项模板未切换。
- 旧项目未迁移。
- 桌面和移动端布局不重叠、不溢出。
