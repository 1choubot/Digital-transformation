# 日报周报 Excel 模板映射

## 个人日报模板

- 文件：`E:\Digital-transformation\docs\项目工作日报-陈芋如20260616.xlsx`
- 工作表：`项目管理工作日报汇总`
- 有效区域：`A1:G21`
- 打印区域：`B2:G14`
- 关键合并区域：`B2:G2`、`B3:G3`、`B5:G5`、`C10:G10`、`B11:G11`

| 业务字段 | 模板位置 | 规则 |
|---|---|---|
| 标题 | `B2:G2` | 固定为项目工作日报 |
| 项目名称 | `B4:C4` | 写入当前日报关联项目名称或编号名称组合 |
| 报告人 | `D4:E4` | 写入当前登录用户姓名 |
| 报告日期 | `G4` | 写入 `YYYY.MM.DD` |
| 今日完成明细 | `B7:G9` | 预留 3 行，超出时复制相邻明细行样式、边框和行高 |
| 进展照片区 | `B10:G10` | P0 先保留区域，图片插入策略在导出器阶段实现 |
| 明日计划明细 | `B13:G14` | 预留 2 行，超出时复制相邻明细行样式、边框和行高 |

## 中心日报模板

- 工作表：`项目管理工作日报汇总`
- 有效区域：`A1:F59`
- 打印区域：`A1:F52`
- 关键合并区域：`A1:F1`、`A3:F3`、`A16:F16`、`A40:F40`、`A53:F55`

|---|---|---|

| 报告人 | `C2:D2` | 写入中心负责人或生成账号展示名 |

| 昨日工作计划 | `A5:F15` | 按员工/项目聚合，超出时复制明细行样式 |
| 今日完成情况 | `A18:F39` | 只聚合已提交日报，草稿不进入 |
| 明日工作计划 | `A42:F52` | 按员工/项目聚合，超出时复制明细行样式 |
| 空态/备注 | `A53:F55` | 无已提交日报时写入“暂无已提交日报” |

## 个人周报模板

- 文件：`E:\Digital-transformation\docs\周绩效考核表-研发中心陈芋如20260621.xlsx`
- 工作表：`Sheet1`
- 有效区域：`A1:H24`
- 打印区域：未设置
- 关键合并区域：`A1:H1`、`A3:H3`、`A5:A15`、`A16:A23`、`A24:H24`

| 业务字段 | 模板位置 | 规则 |
|---|---|---|
| 姓名 | `B2` | 写入当前用户姓名 |
| 部门 | `D2` | 写入用户中心中文名 |
| 岗位 | `F2` | 写入 `users.job_title`，为空时留空 |
| 考核周期 | `H2` | 写入 `YYYY.M.D-YYYY.M.D` |
| 工作总结 | `B5:H15` | 预留 11 行，超出时复制明细行样式 |
| 工作计划 | `B16:H23` | 预留 8 行，超出时复制明细行样式 |
| 签字区 | `A24:H24` | 替换被考评人姓名，保留签字文本布局 |

## 通用导出规则

- 模板文件只作为只读输入，导出时复制或加载模板并另存目标文件。
- 导出时必须替换或清空模板中的样例姓名、日期、项目和明细内容。
- 数据少于预留行时清空多余样例值但保留版式。
- 数据多于预留行时复制相邻明细行的完整样式、边框、行高和合并策略。
- 单元格坐标只能沉淀在导出 builder 或模板映射模块中，不能散落在控制器中。
