# 9.25 旧模板合同审核记录补入项目工作区产出卡片规划

日期：2026-07-02

## 1. 背景

`audit-workspace-card-coverage-before-legacy-checklist-cleanup-v1` 完成后，当前运行 20260625 64 项资料的 workspace card 覆盖率为 62/64。剩余未覆盖资料为：

- `3.3 合同审核记录表（销售合同）`
- `5.4 采购合同审核记录表`

这两项属于当前运行 64 项资料，但在 v20260629 目标模板中不再作为 71 项目标产出保留。因此它们需要作为旧模板兼容资料补入上方项目工作区产出卡片，而不是重新加入 71 项目标模板。

## 2. 当前目标

本 change `map-legacy-contract-review-documents-to-workspace-cards-v1` 的目标是为 `3.3`、`5.4` 补充上方 workspace card 主入口，使当前运行 64 项资料覆盖率达到 64/64。

本 change 明确不是：

- 不是切换 v20260629 71 项模板。
- 不是迁移旧项目或补初始化旧项目。
- 不是删除、隐藏或折叠旧资料清单。
- 不是新增合同审核流、采购审核流或流程引擎。
- 不是数据库 schema 调整，不写 migration。

## 3. 设计口径

- 绑定方式：`legacyDocumentCode=3.3`。
- 目标模板语义：`targetTemplateStatus=removed_in_v20260629`，不计入 71 项目标模板。

### 5.4 采购合同审核兼容项

- 阶段：生产制作阶段。
- 蓝色模块：采购合同审核记录兼容项。
- 关联语境：签订采购合同附近。

- 绑定方式：`legacyDocumentCode=5.4`。
