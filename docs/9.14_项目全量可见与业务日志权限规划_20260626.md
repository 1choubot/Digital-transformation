# 9.14 项目全量可见与业务日志权限规划

日期：2026-06-26

## 目标

本规划用于 `align-project-visibility-and-audit-access-v1`，目标是按最终口径放宽“查看权限”，但不放宽“业务操作权限”。

## 查看权限口径

1. 总经理、总经理助理、中心负责人可查看全部项目、全部阶段资料、全部已上传附件和全部业务日志。
2. 项目创建人可查看自己创建项目的项目详情、全部阶段资料、全部已上传附件和业务日志。
3. 项目经理继续可查看自己负责项目的全部阶段资料、全部已上传附件和业务日志。
4. 员工仍只查看自己相关项目和资料。
5. 系统管理员只做账号和平台维护，不默认获得业务项目、阶段资料、附件或业务日志访问权。

## 只放宽查看能力

本 change 只规划放宽以下能力：

- 项目列表、项目详情和项目总览可见性。
- 阶段资料条目可见性。
- 已上传附件查看和下载权限。
- 项目业务日志查看权限。

附件查看和下载属于“全量资料查看”的一部分。全量查看角色可以看到资料项上的已上传附件列表，并可通过在线平台下载这些附件。

## 绝不放宽业务操作

查看权限放宽不等于操作权限放宽。以下能力必须保持既有权限边界：

- 资料提交。
- 资料审核、确认、退回。
- 精准返工退回。
- 责任人分配和清空。
- 适用性标记和恢复。
- 附件上传。
- 附件删除。
- 阶段推进。
- 项目编号填写。
- `1.2 项目立项审批表` 多节点审批。

后续实现不得把查看函数和操作函数合并复用。可调整的查看判断包括 `visibility.js`、`canViewStageDocumentItem`、`canViewStageDocumentAttachments` / `canDownloadStageDocumentAttachment`、`canViewProjectOperationLogs`。`canViewCompleteProjectAudit` 或等价 helper 保留给 legacy 阶段审批历史只读查看，不作为业务日志放宽 helper。不得放宽 `canReviewStageDocument`、`canSubmitStageDocument`、`canManageProjectResponsibility`、`canManageStageDocumentApplicability`、`canAdvanceProjectStage`、精准返工退回权限。

`buildStageDocumentPermissions` 如需调整，只能调整 `canViewAttachments` 和 `canDownloadAttachment`。不得调整 `canUploadAttachment`、`canDeleteAttachment`、`canSubmitDocument`、`canReviewDocument`、`canManageResponsibility`、`canChangeApplicability`。

## 前端展示口径

- 项目列表和项目总览以后端返回的可见项目为准。
- 普通员工仅因资料责任关系可见项目总览卡片时，只展示项目基础卡片、个人待办数和自己可见的信息；不得返回或展示其他资料编号、名称、状态、`completionMode` 等明细。
- 完整当前阶段齐套摘要和完整未完成资料明细只给总经理、总经理助理、中心负责人、项目创建人、项目经理等完整资料查看角色。
- 项目详情中，全量查看角色可看到完整阶段资料清单、附件列表、下载入口和业务日志。
- 项目详情是页面体验，不等于 `GET /api/projects/:projectId` 单接口承载全部数据；项目基础状态接口只返回项目主数据、8 阶段基础状态和创建人追溯字段，阶段资料、附件列表/下载、业务日志、齐套摘要由阶段资料、附件、日志、总览/齐套等独立接口组合展示。
- 操作按钮仍以后端权限字段为准，不因项目可见或资料可见而展示提交、审核、退回、精准返工、责任人分配、适用性、上传、删除、阶段推进或项目编号填写入口。
- 系统管理员不因能进入用户管理而看到业务项目、业务日志或业务附件。

## 非目标

本 change 不做以下事项：

- 不实现业务代码。
- 不改数据库或 migration。
- 不做日报、周报、项目一览表生成。
- 不实现 `1.2 项目立项审批表` 商务评价、技术评价、总经理多节点在线审批。
- 不做文件平台联动，不调用文件管理平台 API。
- 不做通用审批流引擎，不做推送通知。

## 后续验证重点

- 中心负责人能看跨中心项目、完整 64 项资料、业务日志和附件下载。
- 总经理助理能看全部项目、完整 64 项资料、业务日志和附件下载。
- 项目创建人能看自己创建项目的完整资料和业务日志。
- 系统管理员仍看不到业务项目和业务日志。
- 员工仍只能看自己相关项目和资料。
- 跨中心中心负责人、总经理助理若不是既有授权主体，仍不能审核、提交、退回、精准返工、分配责任人、标记适用性、推进阶段、上传附件或删除附件。
