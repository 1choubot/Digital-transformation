# 模板数字化方案与字段映射草案

版本：v0.2  
日期：2026-05-17  
范围：`智能制造项目管理文件模板/` 目录下 17 个真实模板文件

## 1. 编制原则

- 本草案只依据模板文件中真实出现的 sheet、表格、单元格标签、表头、段落和幻灯片文本整理。
- “疑似填写字段”只表示模板中存在可填写位置或表格列，不代表必填项；必填规则需要业务确认。
- 未出现在模板里的字段不在本草案中补充。
- 第一版数字化方式分为三类：适合在线表单、混合处理、仅文件上传。
- 对复杂公式、复杂版式、PPT、正式签字扫描件等资料，第一版建议保留文件上传，不强行拆成在线表单。
