### P0：1.1 责任人分配入口对营销中心负责人不可操作

- 账号：`marketing_manager`
- 项目：`578 / WORKSPACE_REGRESSION_BASE_20260701132800`
- 操作：
  1. 登录 `marketing_manager`
  2. 进入 `/projects/578`
  3. 点击“项目市场调研”节点
  4. 查看 1.1 项目需求表上方产出卡片
- 预期：营销中心负责人可以分配 1.1 项目需求表责任人，并可看到在线表单入口。
- 实际：在线表单入口存在，但责任人 select 和“保存责任人”按钮禁用，页面提示“当前账号无权分配该资料责任人。”。
- 影响：立项阶段 1.1 的业务前置依赖“营销中心负责人分配责任人”，当前 UI/权限结果不满足测试目标。
- 初步判断：1.1 默认责任中心为研发中心、默认审核中心为营销中心，当前权限可能按责任中心授予分配权，未覆盖“营销中心负责人组织调研并分配 1.1 责任人”的业务规则。
