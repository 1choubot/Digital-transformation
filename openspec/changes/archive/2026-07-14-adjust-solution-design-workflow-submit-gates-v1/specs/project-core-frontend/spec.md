## ADDED Requirements

### Requirement: 方案设计提交门禁调整前端
项目核心前端 MUST 按后端返回的状态和权限展示方案设计重提、财务审批分支选择和产出无需上传交互。

#### Scenario: 退回后可直接重提
- **WHEN** 方案设计内部节点被退回
- **AND** 后端返回旧 current 文件仍满足重提门禁
- **THEN** 前端 MUST 展示可直接重提入口
- **AND** 前端 MUST 继续允许用户重新上传文件覆盖旧文件
- **AND** 前端 MUST NOT 文案提示用户必须重复上传同一文件

#### Scenario: 财务成本估算总经理审批时选择分支
- **WHEN** 总经理在前端审批通过财务成本估算
- **THEN** 前端 MUST 要求选择报价流程或投标流程
- **AND** 前端 MUST 将分支选择随审批通过请求提交给后端
- **AND** 未选择分支时前端 MUST 阻止提交或展示后端业务错误

#### Scenario: 报价投标节点显示已选分支
- **WHEN** 前端展示报价/投标节点
- **AND** 后端已返回财务总经理审批时选择的分支
- **THEN** 前端 MUST 直接展示报价流程或投标流程
- **AND** 前端 MUST NOT 再展示重复的分支选择主入口

#### Scenario: 单项产出无需上传入口
- **WHEN** 技术负责人处理方案设计 8 个产出
- **THEN** 前端 MUST 对每个产出展示后端允许的“无需上传”入口
- **AND** 前端 MUST 支持填写原因或备注
- **AND** 前端 MUST 展示豁免状态、操作人和时间
- **AND** 前端 MUST 允许仍需上传的产出继续使用上传入口

#### Scenario: 豁免产出不阻塞提交展示
- **WHEN** 某方案设计产出已标记无需上传
- **THEN** 前端 MUST 将该产出显示为已满足或等价不阻塞状态
- **AND** 前端 MUST NOT 为该产出继续提示必须上传文件
- **AND** 未上传且未豁免的产出 MUST 继续显示为阻塞提交

#### Scenario: 重新上传后取消豁免展示
- **WHEN** 某方案设计产出已标记无需上传
- **AND** 用户对该产出重新上传文件且后端返回豁免已取消
- **THEN** 前端 MUST 展示新上传文件为该产出的 current file
- **AND** 前端 MUST 不再展示该产出仍处于无需上传豁免状态
