## 1. 技术栈确认

- [x] 1.1 确认正式前端技术栈为 Vue + Vite
- [x] 1.2 确认正式后端技术栈为 Node.js + Express
- [x] 1.3 确认正式数据库为 MySQL
- [x] 1.4 确认前端 Demo 只作为页面参考，不作为正式数据架构

## 2. 服务与数据库边界

- [x] 2.1 确认数字化平台作为独立服务端运行
- [x] 2.2 确认数字化平台使用独立 MySQL 数据库或独立 MySQL schema
- [x] 2.3 确认数字化平台不直接共用文件管理平台数据库
- [x] 2.4 确认数字化平台不能直接读写文件管理平台数据库表

## 3. 文件与平台联动边界

- [x] 3.1 确认正式文件仍由文件管理平台存储
- [x] 3.2 确认数字化平台只保存业务数据、状态和必要文件平台标识
- [x] 3.3 确认两平台通过 HTTP API 联动
- [x] 3.4 确认数字化平台不通过共享内部模块或直接操作文件目录绕过文件平台 API

## 4. 部署口径

- [x] 4.1 确认数字化平台第一版为内网 Web 系统
- [x] 4.2 确认数字化平台和文件管理平台同服务器部署
- [x] 4.3 确认两套平台保持独立端口、配置和服务进程边界
- [x] 4.4 确认文件管理平台 HTTP API 基础地址和调用凭据通过配置项管理

## 5. 架构验收

- [x] 5.1 检查 proposal、design 和 spec 是否一致表达 Vue + Vite、Node.js + Express、MySQL 架构
- [x] 5.2 检查架构文档是否明确禁止直接共用或读写文件管理平台数据库
- [x] 5.3 检查架构文档是否明确文件仍由文件管理平台存储
- [x] 5.4 检查架构文档是否明确两平台通过 HTTP API 联动
- [x] 5.5 检查架构文档是否明确 Demo 只作为页面参考
